Users â–¼

List all users:
Get-MgUser -All | Select-Object DisplayName, UserPrincipalName, Id
Get a specific user:
On-prem synced users (UPNs):
Get-MgUser -Filter "onPremisesSyncEnabled eq true" -All | Select-Object -ExpandProperty UserPrincipalName

Groups â–¼

List all groups:
Get-MgGroup -All | Select-Object DisplayName, Id, GroupTypes
Members of a group:
Get-MgGroupMember -GroupId "<group-id>" | Select-Object Id, AdditionalProperties

Directory Roles â–¼

List active directory roles:
Get-MgDirectoryRole -All | Select-Object DisplayName, Id
Members of a role (e.g. Global Administrator):
Get-MgDirectoryRoleMember -DirectoryRoleId "<role-id>"
