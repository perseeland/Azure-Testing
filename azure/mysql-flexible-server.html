MySQL Flexible Server

Overview ▼

  • What it is — managed MySQL database (the modern alternative to "Single Server").
  • Networking — may allow public access via firewall rules; check for overly broad ranges (e.g. 0.0.0.0-255.255.255.255).
  • Auth — MySQL native (username/password) and/or Microsoft Entra logins.
  • Creds — look in Container App / App Service env vars, Key Vault secrets, Automation Account variables, or ARM templates.
  • ARM roles (Reader/Contributor) only manage the server resource (e.g. firewall) — valid MySQL credentials are needed for database access.
