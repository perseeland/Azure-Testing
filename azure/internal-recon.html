Identify and Set the Subscription ID â–¼

List subscriptions with the ARM token and capture the subscription ID:
SubscriptionID=$(curl -s -X GET \ "https://management.azure.com/subscriptions?api-version=2025-04-01" \ -H "Authorization: Bearer $ARMAccessToken" \ -H "Content-Type: application/json" \ | jq -r '.value[].subscriptionId') && echo $SubscriptionID

Identify Resources â–¼

List resources in the subscription (type and name):
curl -s -X GET \ "https://management.azure.com/subscriptions/$SubscriptionID/resources?api-version=2021-04-01" \ -H "Authorization: Bearer $ARMAccessToken" \ -H "Content-Type: application/json" \ | jq -r '.value[] | {Type: .type, Name: .name}'
Show the resource group for each resource (the RG is the 4th segment of the resource id):
curl -s -X GET \ "https://management.azure.com/subscriptions/$SubscriptionID/resources?api-version=2021-04-01" \ -H "Authorization: Bearer $ARMAccessToken" \ -H "Content-Type: application/json" \ | jq -r '.value[] | {Name: .name, Type: .type, RG: (.id | split("/")[4])}'

List storage accounts in the subscription:
curl -s -X GET \ "https://management.azure.com/subscriptions/$SubscriptionID/providers/Microsoft.Storage/storageAccounts?api-version=2023-01-01" \ -H "Authorization: Bearer $ARMAccessToken" \ -H "Content-Type: application/json" \ | jq -r '.value[].name'
