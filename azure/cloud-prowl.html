Cloud Prowl

CloudProwl takes an Entra ID refresh token and enumerates which Azure / Microsoft 365 services and APIs that token can reach — quickly mapping the access a stolen token grants.

Run CloudProwl ▼

Clone the repo:
git clone https://github.com/pwnedlabs/cloudprowl.git
Set the refresh token:
Run it against the token:
python3 cloudprowl.py "$refreshToken"
