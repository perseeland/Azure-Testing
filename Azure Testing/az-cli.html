AZ CLI

User Login ▼

Interactive login (browser):
az login
Log in with a username and password:
Get an access token for Microsoft Graph:
az account get-access-token --resource https://graph.microsoft.com
Get an access token for ARM (management):
az account get-access-token --resource https://management.azure.com

Service Principal Login ▼

Microsoft Graph API — SPN Access Token (application-level permission). Authenticates as the service principal itself; permissions come from the app's granted roles.

List Resources ▼

Name, group, type & tags (table):
az resource list --query "[].{name:name,resourceGroup:resourceGroup,types:type,tagg:tags}" -o table

Role Assignments ▼

Role assignments for the signed-in user (all scopes):
az role assignment list --assignee $(az ad signed-in-user show --query id -o tsv) --all -o table
Role & Scope Summary for the signed-in user (role + scope only):
az role assignment list --assignee $(az ad signed-in-user show --query id -o tsv) --all --query "[].{role:roleDefinitionName, scope:scope}" -o table
