Sites

Overview ▼

  • Azure App Service (Web Apps) — hosts web apps, REST APIs, and mobile backends. Enumerating a site's configuration can reveal app settings (secrets), connection strings, deployment credentials, and a managed identity.
  • Publishing profiles — contain plaintext deployment credentials (FTP/MSDeploy) that can be used to pull or push code to the site.
  • SCM / Kudu — the source control management endpoint (https://<app>.scm.azurewebsites.net) exposes a console, file browser, and deployment API for the site.
To access the SCM (Kudu) endpoint for this site, the Website Contributor role must be assigned first — without it, requests to https://<app>.scm.azurewebsites.net will be denied.
