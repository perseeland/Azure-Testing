AZ PowerShell

Connect with an Access Token â–¼

Import the Az module (or install it first if missing):
Import-Module Az
Install-Module Az -Force -AllowClobber
Set your access token:
Authenticate with the token:
Connect-AzAccount -AccessToken $AccessToken -AccountID "Anything"
Log out when done:
Disconnect-AzAccount

Get Access Token â–¼

ARM token (default resource) from the current session:
(Get-AzAccessToken).Token
Management (ARM) token:
(Get-AzAccessToken -ResourceUrl "https://management.azure.com/").Token
Microsoft Graph token:
(Get-AzAccessToken -ResourceUrl "https://graph.microsoft.com/").Token

List Resources â–¼

Enumerate resources (group, type, name, tags) sorted by type:
Get-AzResource | Select-Object ResourceGroupName,ResourceType,Name,Tags | Sort-Object ResourceType
Detailed table â€” tags flattened to key=value pairs:
Get-AzResource | Select-Object ResourceGroupName, ResourceType, Name, @{N='Tags'; E={ if ($_.Tags) { ($_.Tags.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join '; ' } else { '' } }} | Sort-Object ResourceGroupName, ResourceType, Name | Format-Table -AutoSize
Grouped by resource group (-GroupBy):
Get-AzResource | Select-Object ResourceGroupName, ResourceType, Name, @{N='Tags'; E={ if ($_.Tags) { ($_.Tags.GetEnumerator() | ForEach-Object { "$($_.Key)=$($_.Value)" }) -join '; ' } else { '' } }} | Sort-Object ResourceGroupName, ResourceType | Format-Table -AutoSize -GroupBy ResourceGroupName
