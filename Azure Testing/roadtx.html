roadtx

roadtx (ROADtools Token eXchange) authenticates to Entra ID and requests/exchanges tokens — e.g. trading a refresh token for access tokens scoped to different clients and resources.

Refresh Token → Access Token ▼

Set the refresh token:
Exchange it for an ARM access token (client = Azure CLI) and print to stdout:
roadtx auth --refresh-token "$refreshToken" -c "04b07795-8ddb-461a-bbee-02f9e1bf7b46" -r "https://management.core.windows.net/" --tokens-stdout
