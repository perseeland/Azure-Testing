Get an Azure AD Graph token (ROPC, scoped to graph.windows.net):
Why the AAD Graph token? roadrecon enumerates the directory through the legacy Azure AD Graph API (graph.windows.net), which still exposes Conditional Access Policies and other objects ├втВмтАЭ so the token must be scoped to that resource, not Microsoft Graph.
Authenticate roadrecon with the token:
roadrecon auth --access-token $aadgraphtoken
Gather the directory data:
roadrecon gather
Dump the policies (Conditional Access):
roadrecon plugin policies
