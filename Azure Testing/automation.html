Automation

Overview ▼

  • Azure Automation Account — hosts runbooks, jobs, credentials, and variables for automating Azure infrastructure tasks.
  • Runbooks — PowerShell or Python scripts that run on a schedule or manually to perform automated tasks.
  • Jobs — execution records of runbooks, with logs and output that can reveal sensitive information or expose automation failures.
  • Permissions — RBAC determines whether an attacker can read/execute runbooks, modify credentials, or exfiltrate job output.
